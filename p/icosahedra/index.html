Icosahedra
From Fakepedia, the made up encyclopedia

Icosahedra (automatus icosadrus) is a class of robots that were created by Hexahedra
They are primarily utilized on discord.

First generation
The first generation of icosahedra was developed using JavaScript.
Its features were primitive as hexahedra were still researching the language

Later generations
The second generation and up were built with C#
The features of these are lost to time.

Latest generations
The second last generation attempted to include the game of Battleship, but was inevitably lost in an SSD failure event.
The last generation is lacking in any notable features.
